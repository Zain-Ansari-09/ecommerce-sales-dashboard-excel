# Ecommerce Sales Performance Dashboard (Excel)

## Project Overview

This project presents an interactive **Ecommerce Sales Performance Dashboard** built using Microsoft Excel. The dashboard analyzes sales performance from **January to April 2024** and provides insights into revenue, product performance, customer location, and payment methods.

## Key Metrics

* Total GMV (Gross Merchandise Value)
* Total Orders
* Units Sold
* Average Order Value (AOV)

## Dashboard Features

* Sales trend analysis
* Product category performance
* Top products by revenue
* Regional sales distribution
* Payment method breakdown
* Interactive filters using slicers

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization
* Slicers for interactivity

## Key Insights

* Electronics category drives the highest revenue.
* Card payments account for 52% of transactions.
* Los Angeles is the top revenue generating city.
* Smartphones are the best performing product.

## Data Source

Simulated Ecommerce Sales Dataset created for analytics practice.
